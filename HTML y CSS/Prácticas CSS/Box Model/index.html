Para acordanos donde va cada valor, pensar en el sentido de las aguajs del reloj. (También sirve para el margin). Si ponemos solo dos valores, el primero corresponde a arriba y abajo, y el segundo a los costados. Si hay tres valores, es arriba, costados y abajo.
